# Backend Case Study - Bynry

## Overview
This project implements a backend API for low-stock alerts in a B2B inventory system.

## Tech Stack
- Node.js
- Express.js

## API Endpoint
GET /api/companies/:company_id/alerts/low-stock

## Features
- Fetch low stock products
- Supports multi-warehouse logic (conceptual)

## Assumptions
- Threshold predefined
- Dummy data used

## How to Run
npm install
node app.js

## info
This implementation demonstrates backend API design, low-stock alert logic, and scalable system thinking for a multi-warehouse inventory system.
